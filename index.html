linux-indexer

linux local filesystem indexer with web frontend.

View project on GitHub

Welcome

Linux indexer serves a specific purpose for users that have a lot of media, including multiple copies of backups, dispersed across many filesystems. It provides user an elegant web interface to browse indexed filesystems and provides variety of features to manage the indexed data. For example, it can easily show you that there are three identical files with different names and attributes spread across different filesystems and remove copies.
